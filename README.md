# 可视化停车场管理系统

一个基于 Qt 的可视化停车场管理系统，支持管理员和普通用户角色，提供车辆管理、计费服务、自动驶入等功能。

## 功能特性

### 管理员功能
- **车辆管理**：添加、删除车辆，支持自定义车牌号、车型、颜色和图片
- **车辆驶入/驶出**：支持手动驶入和自动驶入模式
- **智能寻路**：自动驶入模式下优先选择靠近入口的空位
- **计费服务**：支持首小时费率、额外小时费率、每日最高收费、免费时长等配置
- **车辆详情**：点击车辆可查看详细信息（停车位置、进入时间、停车时长、费用等）
- **信息修改**：支持修改已停车车辆的信息

### 用户功能
- **停车场查看**：查看停车场实时状态
- **登录系统**：普通用户登录查看停车场信息

### 系统设置
- **停车场配置**：可自定义停车场行数和列数
- **计费规则设置**：灵活配置计费参数

## 技术架构

- **语言**：C++

- **构建工具**：CMake

## 编译运行

### 环境要求
- Qt 6.x（推荐 6.10.2）
- CMake 3.16+
- MinGW 或 MSVC 编译器

### 编译步骤

1. 打开 Qt Creator
2. 打开项目目录下的 `CMakeLists.txt`
3. 选择构建配置（Debug/Release）
4. 点击构建按钮

### 运行

编译成功后，运行生成的可执行文件 `VisualParkingManagementSystem.exe`。

## 使用说明

### 登录系统
1. 打开应用程序
2. 点击菜单「登录」
3. 选择登录类型（管理员/用户）
4. 输入用户名和密码
   - 默认管理员：用户名 `stay`，密码 `123456`

### 管理员操作

#### 添加车辆
1. 点击菜单「管理」->「打开管理界面」
2. 在管理界面中输入车牌号、车型
3. 选择车辆颜色（可选）
4. 选择车辆图片（可选）
5. 点击「添加车辆」

#### 车辆驶入
- **手动模式**：点击「车辆进入(顺序)」，车辆从等待队列依次进入停车场
- **自动模式**：点击「车辆进入（auto）」，系统每1.5秒自动驶入一辆车

#### 车辆驶出
- 点击「车辆驶出」，车辆按进入顺序依次驶出

#### 查看车辆详情
- 点击停车场内的任意车辆，弹出详情对话框
- 显示车辆信息、停车位置、进入时间、停车时长和当前费用

### 系统设置
1. 点击菜单「停车场设置」
2. **停车场设置**：设置停车场的行数和列数
3. **计费规则设置**：
   - 首小时费率：第一小时的收费标准
   - 额外小时费率：超过一小时后的收费标准
   - 每日最高收费：单日最高收费上限
   - 免费时长：停车免费时长

## 计费规则

计费公式：
```
停车时长 <= 免费时长 → 费用 = 0
停车时长 <= 1小时 → 费用 = 首小时费率
停车时长 > 1小时 → 费用 = 首小时费率 + (小时数 - 1) × 额外小时费率
最终费用 = min(计算费用, 每日最高收费)
```

## 配置文件

- `name_password.txt`：用户账号密码存储
- `fee_rule.txt`：计费规则配置
